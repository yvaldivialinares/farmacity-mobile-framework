### Description
Provides a back navigation control to return to the user-login-screen. Uses resource-id `header-back-text` — a different pattern from the `icon-button` resource-id used as the back button on all other mapped screens.

The distinct resource-id (`header-back-text` vs `icon-button`) means this back button cannot share a locator with other screens' back buttons in a shared BasePage method.
